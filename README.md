# Learning Experiment

A public record of AI-assisted PhD work at Texas State University.

## What This Is

I'm a doctoral student and student affairs practitioner using Claude (Anthropic) as a
daily collaborator — research analysis, coursework, writing, infrastructure, everything.
This repo documents that experiment transparently.

## How to Navigate

- **reflections/** — Curated writing about what I'm learning. The narrative layer.
- **sessions/** — Raw Claude Code session archives. The process layer. Prompts, decisions, iterations.
- **Rendered site** — [this15fine.dev/learning/](https://this15fine.dev/learning/) — these same files, rendered with proper formatting and a changelog.

## Why It's Public

Transparency is the point. If AI collaboration is legitimate scholarly practice — and I
believe it is — then the process should be visible. Every reflection links to its source.
Every changelog entry links to a commit. The connection between the site and this record
is never hidden.
